## Comments

You can add comments to your code using the `#` symbol. It can go at the
beginning of a line, or in the middle of a line after whatever code is
on that line.

``` r
# a full line comment
1 + 1
```

    ## [1] 2

``` r
2 + 2 # comment immediately after code
```

    ## [1] 4
